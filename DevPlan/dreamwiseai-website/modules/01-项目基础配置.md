# 模块 01 — 项目基础配置

## 模块概述

| 属性 | 值 |
|------|-----|
| 类型 | Foundation |
| 优先级 | P0 |
| 预估工时 | 2h |
| 依赖 | 无 |

## 功能清单

1. Next.js standalone 模式配置（Docker 必须）
2. Tailwind 品牌色和字体 CSS 变量
3. Google Fonts（Sora + Inter）通过 next/font 加载
4. 全局 CSS 重置
5. 依赖包安装确认

## 开发步骤

### Step 1: next.config.mjs

```javascript
import createNextIntlPlugin from 'next-intl/plugin'

const withNextIntl = createNextIntlPlugin()

/** @type {import('next').NextConfig} */
const nextConfig = {
  output: 'standalone',  // K8s Docker 部署必须
}

export default withNextIntl(nextConfig)
```

### Step 2: tailwind.config.ts

```typescript
import type { Config } from 'tailwindcss'

const config: Config = {
  content: [
    './app/**/*.{js,ts,jsx,tsx,mdx}',
    './components/**/*.{js,ts,jsx,tsx,mdx}',
  ],
  theme: {
    extend: {
      colors: {
        brand: {
          indigo: '#1E3A8A',
          blue: '#3B82F6',
        },
      },
      fontFamily: {
        sora: ['var(--font-sora)', 'sans-serif'],
        inter: ['var(--font-inter)', 'sans-serif'],
      },
    },
  },
  plugins: [],
}

export default config
```

### Step 3: 字体配置（app/fonts.ts）

```typescript
import { Sora, Inter } from 'next/font/google'

export const sora = Sora({
  subsets: ['latin'],
  variable: '--font-sora',
  display: 'swap',
  weight: ['400', '600', '700', '800'],
})

export const inter = Inter({
  subsets: ['latin'],
  variable: '--font-inter',
  display: 'swap',
  weight: ['400', '500', '600'],
})
```

### Step 4: globals.css

```css
@tailwind base;
@tailwind components;
@tailwind utilities;

@layer base {
  html {
    scroll-behavior: smooth;
  }

  body {
    @apply font-inter text-slate-900 bg-white;
    -webkit-font-smoothing: antialiased;
  }

  h1, h2, h3, h4, h5, h6 {
    @apply font-sora;
  }
}
```

### Step 5: 安装依赖

```bash
npm install next-intl resend
# framer-motion, lucide-react 已安装
```

### Step 6: .env.local.example

```bash
# Resend

# Calendly

```

## 验收标准

- `npm run dev` 启动无报错
