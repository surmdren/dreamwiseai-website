# 成本估算

## K8s 自托管成本

| 项目 | 服务 | 月费用 | 说明 |
|------|------|--------|------|
| VPS 服务器 | Hetzner CX22 / Vultr 1C2G | $6-12/月 | 运行 k3s 集群 |
| 域名 | Namecheap (已有) | ~$12/年 ≈ $1/月 | dreamwiseai.com |
| TLS 证书 | Let's Encrypt (免费) | $0 | cert-manager 自动续期 |
| 邮件发送 | Resend 免费套餐 | $0 | 3000 封/月，够用 |
| 预约 | Calendly 免费套餐 | $0 | 1 个事件类型 |
| 镜像仓库 | Docker Hub 免费 / GitHub Container Registry | $0 | 公开镜像 |

**合计：约 $7-13/月**

## 与 Vercel 对比

| 方案 | 月费用 | 优点 | 缺点 |
|------|--------|------|------|
| Vercel + Supabase | $0（MVP 阶段）| 零运维，分钟上线 | 流量大后付费，数据不自控 |
| **K8s 自托管（当前选择）** | **$7-13/月** | **数据完全自控，可扩展** | 需要服务器管理知识 |

## 推荐服务器

| 服务商 | 规格 | 价格 | 特点 |
|--------|------|------|------|
| Hetzner | CX22 (2C4G) | €4.5/月 | 欧洲，性价比最高 |
| Vultr | 1C2G | $6/月 | 全球节点，中国访问相对快 |
| DigitalOcean | 1C2G Droplet | $6/月 | 文档好，新手友好 |
| 阿里云 ECS | 1C2G | ¥50-80/月 | 中国访问最快，适合主要用户在国内 |

**建议**：若目标用户在中国，选阿里云 ECS；若全球用户，选 Hetzner 或 Vultr。
