# DreamWise AI 官网 — UIUX 设计文档总览

## 项目设计概述

DreamWise AI 官网，单页 Landing Page，12 个区块，目标是将中小企业决策者转化为咨询预约客户。

## 目标用户画像

- **主要**：中小企业 CEO/运营总监/IT 负责人，40-55 岁，非技术背景，关注 ROI 和风险
- **次要**：对 AI 转型感兴趣的业务负责人，30-45 岁

## 设计原则

1. **信任优先**：每个区块都在传递专业感和可信度
2. **转化导向**：双 CTA（预约 + 留资），路径清晰
3. **克制动效**：fade-in 辅助，不喧宾夺主
4. **双语无缝**：中英文切换不影响布局
