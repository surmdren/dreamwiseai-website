# DreamWise AI 官网 - 初始需求

## 原始需求

重建 DreamWise AI 咨询公司官网，从零开始。

## 需求文档

完整需求见：`docs/specs/2026-03-17-website-redesign.md`（v1.5，已确认）

## 核心目标

帮 Rick 创建一个 24/7 自主运营的公司，在一年内通过 AI 驱动的产品、服务和内容，实现年收入 100 万美元。官网是获客入口的第一步。

## 页面结构（12 个区块）

1. Navbar — Logo + 导航 + Book a Free Call，粘性顶部，中英切换
2. Hero — 大标题 + slogan + 双CTA + 抽象SVG背景
3. Trust Logos — "Trusted by" logo 墙（占位）
4. Problem — 3个中小企业AI转型痛点卡片
5. Services — Assessment / Development / Operations
6. How It Works — Assess → Build → Operate 三步流程
7. Case Studies — 3个案例卡片（占位）
8. Stats — 4个关键数字
9. CTA — Calendly预约（主）+ 留资表单（备选）+ Resend邮件通知
10. Testimonials — 2-3个客户证言（占位）
11. About — 团队背景 + 创立原因
12. Footer — Logo + 链接 + 版权

## 品牌

- 公司名：DreamWise AI
- Slogan：The Future is Agent-Driven. We'll Get You There.
- 主色：Deep Indigo #1E3A8A + Bright Blue #3B82F6
- 风格：Clean Professional，大量留白

## 非功能性需求

- 中英双语（自动检测 + 手动切换）
- 响应式（Mobile / Tablet / Desktop）
- SEO（双语 meta + og:image）
- 表单：Resend 邮件通知
- 预约：Calendly inline widget
- 动效：适度 scroll fade-in
